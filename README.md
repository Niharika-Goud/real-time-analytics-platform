# Real-Time Analytics Platform

A scalable real-time analytics platform built using FastAPI, React.js, PostgreSQL, Redis, Celery, WebSockets, and Docker.

## Features
- JWT Authentication
- Real-Time Dashboard
- WebSocket Updates
- Redis Caching
- Celery Background Tasks
- Report Generation
- Dockerized Deployment

## Tech Stack
### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Redis
- Celery

### Frontend
- React.js
- Recharts

### DevOps
- Docker
- Docker Compose

## Run Project

```bash
docker-compose up --build
```
